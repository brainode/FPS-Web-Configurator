# games/warsow

Каталог хранит игровые артефакты и метаданные Warsow.

## Структура

- `distribution/` - исходный дистрибутив Warsow, используемый при сборке `services/warsow-server/`

## Зачем это вынесено отдельно

Так сервисный код не смешивается с файлами самой игры:

- Dockerfile и entrypoint лежат в `services/warsow-server/`
- игровые бинарники, `.pk3` и базовые конфиги лежат в `games/warsow/distribution/`

Это соответствует выбранной гибридной архитектуре: сервисная структура каталогов плюс модульность под разные игры.
