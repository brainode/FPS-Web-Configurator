# services

Каталог `services/` хранит исполняемые сервисы проекта.

Текущий состав:

- `warsow-server/` - runtime-модуль выделенного сервера Warsow;
- `docker-agent/` - будущий локальный агент для безопасного управления Docker;
- `control-panel/` - веб-панель управления на ASP.NET Core.

Подход гибридный:

- структура каталогов сервисная;
- логика конкретных игр в панели будет модульной через адаптеры.
